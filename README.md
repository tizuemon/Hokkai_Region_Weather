# 北海道・樺太・千島 リアルタイム気象マップ

北海道・樺太・千島の各地の現在の天気と予報をリアルタイムで表示するウェブアプリです。

## 特徴
- **広域カバー**: 北海道全域に加え、樺太南部・北部、南千島・中千島・北千島まで網羅。
- **リアルタイム更新**: 5 分おきに自動で最新の気象情報を取得。
- **詳細予報**: 各都市をクリックすると、24 時間の時系列予報と 1 週間の週間予報を表示。

## 使用データ・ライセンス

### 1. 背景地図データ
本プロジェクトの背景地図は、以下のデータを使用・加工して作成しています。
- **出典**: 昭和20(1945)年時点南樺太・北千島市町村Shapeデータ
- **制作者**: 八鬮不動産
- **ライセンス**: [クリエイティブ・コモンズ 表示 4.0 国際 (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/deed.ja)
- **加工の内容**: Web 表示用に画像を軽量化し、各エリアごとの分割、および気象観測地点のプロット用座標調整を行っています。

### 2. 気象データ
- **提供元**: [Open-Meteo.com](https://open-meteo.com/)
- **利用規約**: 非商用利用および個人利用の範囲内で、オープンデータとして利用しています。

## 開発者向け情報
このリポジトリは GitHub Actions を利用したキャッシュサーバー方式を採用しています。

- **`update_weather.py`**: Open-Meteo API から一括でデータを取得する Python スクリプト。
- **`.github/workflows/update.yml`**: 5 分おきにスクリプトを実行し、`weather_data.json` を更新するワークフロー。
- **`weather_data.json`**: フロントエンドが直接読み込むキャッシュされた気象データ。
